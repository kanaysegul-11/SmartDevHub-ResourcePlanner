# SmartDevHub Frontend

Bu klasör SmartDevHub Resource Planner uygulamasının React + Vite arayüzünü içerir.

## Çalıştırma

```powershell
npm install
npm run dev
```

Varsayılan backend API adresi:

```text
http://localhost:8000/api
```

Farklı API adresi gerekiyorsa bu klasörde `.env.local` oluşturun:

```env
VITE_API_BASE_URL=http://localhost:8000/api
```

## Kontroller

```powershell
npm run lint
npm run build
```
